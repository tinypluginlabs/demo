<p class="docs-hubs">The WordPress Playground documentation is distributed across four separate hubs (subsites):</p>

-   [**દસ્તાવેજીકરણ**](/) – WP પ્લેગ્રાઉન્ડનો પરિચય, શરૂઆતની માર્ગદર્શિકાઓ અને WP પ્લેગ્રાઉન્ડ ડોક્સમાં તમારા પ્રવેશ બિંદુ.
-   [**બ્લુપ્રિન્ટ્સ**](/blueprints) – બ્લુપ્રિન્ટ્સ એ તમારા વર્ડપ્રેસ પ્લેગ્રાઉન્ડ ઇન્સ્ટન્સને સેટ કરવા માટે JSON ફાઇલો છે. બ્લુપ્રિન્ટ્સ ડોક્સ હબમાંથી તેમની શક્યતાઓ વિશે જાણો.
-   👉 [**ડેવલપર્સ**](/developers) (તમે અહીં છો) - વર્ડપ્રેસ પ્લેગ્રાઉન્ડ એક પ્રોગ્રામેબલ ટૂલ તરીકે બનાવવામાં આવ્યું હતું. આ ડેવલપર્સ ડોક્સ હબમાં તમારા કોડમાંથી તમે તેની સાથે શું કરી શકો તે બધું શોધો.
-   [**API સંદર્ભ**](/api) – વર્ડપ્રેસ પ્લેગ્રાઉન્ડ દ્વારા ખુલ્લા બધા API

## ડેવલપર્સ દસ્તાવેજીકરણ હબ નેવિગેટ કરવું

<!--
## Navigating the Developers documentation hub
-->

આ ડોક્સ હબ ડેવલપર્સ માહિતી પર કેન્દ્રિત છે અને નીચેના મુખ્ય વિભાગોમાં વિભાજિત થયેલ છે:

<!--
This docs hub is focused on Developers info and is divided into the following major sections:
-->

-   [ડેવલપર્સ માટે ઝડપી શરૂઆત માર્ગદર્શિકા](/developers/build-your-first-app): પ્લેગ્રાઉન્ડ API નો ઉપયોગ કરીને તમે શું પ્રાપ્ત કરી શકો છો તેના કેટલાક ઉદાહરણો શોધીને પ્લેગ્રાઉન્ડ સાથે તમારી વિકાસ યાત્રા શરૂ કરો.

-   [સ્થાનિક વિકાસ](/developers/local-development): વર્ડપ્રેસ સાઇટ્સ સેટઅપ અને મેનેજ કરવાની પ્રક્રિયાને સુવ્યવસ્થિત કરવા અને તમારી એપ્સ બનાવવા માટે પ્લેગ્રાઉન્ડ દ્વારા પૂરા પાડવામાં આવેલા સાધનો શોધો.

-   [પ્લેગ્રાઉન્ડ API](/developers/apis): પ્લેગ્રાઉન્ડ સાથે ક્રિયાપ્રતિક્રિયા કરવા માટે વર્ડપ્રેસ પ્લેગ્રાઉન્ડ દ્વારા ખુલ્લા પાડવામાં આવેલા મુખ્ય API નું અન્વેષણ કરો: [ક્વેરી API](/developers/apis/query-api/)、[બ્લુપ્રિન્ટ્સ](/blueprints/)、[જાવાસ્ક્રિપ્ટ API](/developers/apis/javascript-api/)

-   [સ્થાપત્ય](/developers/architecture): વર્ડપ્રેસ પ્લેગ્રાઉન્ડના આર્કિટેક્ચરની વિશિષ્ટતાઓ, જેમાં તેના વિવિધ ઘટકો અને સાધનોનો સમાવેશ થાય છે.

-   [મર્યાદાઓ](/developers/limitations/): વર્ડપ્રેસ પ્લેગ્રાઉન્ડ ની વર્તમાન મર્યાદાઓ વિશે વધુ જાણો, જેમાં અક્ષમ નેટવર્ક કનેક્શન્સ, કામચલાઉ ડેટા, iframe ક્વિર્ક્સ, PHP ફંક્શન એક્ઝેક્યુશન આવશ્યકતાઓ અને આંશિક WP-CLI સપોર્ટનો સમાવેશ થાય છે.
