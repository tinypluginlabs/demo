# Test a WordPress project with the Playground CLI in an ES Modules project

This is a minimal example of how to test WordPress projects with WordPress Playground using the [Playground CLI](https://github.com/atomicsites/playground-cli) and [Vitest](https://vitest.dev/) to run tests in an ES Modules project.

## Installation

```bash
npm install
```

## Run tests

```bash
npm test
```
